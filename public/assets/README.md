# Assets Directory Structure

This directory contains all static assets for the MovieFinder demo website.

## Directory Guide

### `/images`
Place general images here:
- App logo
- Team member photos
- Background images
- Icons

### `/screenshots`
Place app UI screenshots here:
- Home screen
- Feature screens
- User interface examples
- App workflow images

Recommended format: PNG or WebP
Recommended naming: `screenshot-01.png`, `screenshot-02.png`, etc.

### `/videos`
Place app demo videos here:
- App demo/walkthrough video
- Feature highlights

Recommended format: MP4 (H.264)
Recommended naming: `app-demo.mp4`

### `/apk`
Place the Android APK file here:
- Latest release APK

Recommended naming: `moviefinder-v1.0.apk` or similar with version number

## Notes
- Optimize images before uploading (compress, resize appropriately)
- Keep video file sizes reasonable (under 50MB if possible)
- Use descriptive filenames
